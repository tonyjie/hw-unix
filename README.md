## hw1 - 模拟监控covid-19的物联网
### 编译运行方式
```
cd hw1
make
```
### 参数设置
设定的环境为一个二维地图    
在covid19.hh中通过宏定义规定了一些参数，其含义为：  
X_MIN, X_MAX, Y_MIN, Y_MAX 规定了地图的大小[X_MIN, X_MAX] * [Y_MIN, Y_MAX] = [0, 35] * [0, 35]  
INFECT_DIS 规定 两个人在同一天的距离小于 INFECT_DIS 时，可以互相感染COVID-19    
B_NUM 为簇节点的个数    
P_NUM 为人的数量    
DAY_NUM 为仿真总天数
